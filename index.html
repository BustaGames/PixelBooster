The Project

Pixel::Booster is going to be an easy to use and powerful software for pixel artists. The software is being developed using C++11 and Qt5.5.1. It will feature tools used by known and approved programs used by multiple projects over the past decades, but also modern features to help creating assets for games and animations faster and more comfortably. Pixel::Booster will be entirely free and open source forever. This means the community can improve the software by suggesting new features and fixing bugs! Latest Release: Version 0.03 Alpha

Currently there are only binaries for Windows (x86 and x64). For other platforms, you may download the source code and compile the project. Please, help us to improve the software by reporting any errors at our github page!
