Pixel::Booster

Pixel Art Image Editor using C++ and Qt

The Project

Pixel::Booster is going to be an easy to use and powerful software for pixel artists. Will feature tools used by known and approved programs used by multiple projects over the past decades, but will also contain modern features to help creating assets for games and animations with more comfort and agility. Pixel::Booster is going to be entirely free and open source. This means the community can improve the software by suggesting new features and fixing bugs!

Planned Features

Here are some of the features currently planned for the future releases of the software. The features marked by (✓) are currently implemented in the source code. For the features on each version, check the release notes.

Version 0.0 (Alpha)

  • Create new images ✓
  • Load existing images ✓
  • Transfer image from canvas to edit area ✓
  • Transfer image from edit area to canvas ✓
  • Pencil Tool ✓
  • Color Pick ✓
  • Saving image (png, jpg, bmp) ✓
  • 32 color bit image format only ✓
  • Language Options (Portuguese and English only for now) ✓
  • Setting tile size ✓
  • Toggle Transparency from pick and paste mode. ✓
  • Save app state through multiple executions. ✓
  • Color Palette

Version 0.1 (Alpha)

  • Line Tool
  • Ellipse Tool
  • Rectangle Tool
  • Bucket Tool
  • Text Tool
  • Zoom
  • Selection Tool
  • Copy
  • Paste

Version 0.2 (Alpha)

  • Show grid
  • Show Divisions
  • Change Image Size
  • Resize
  • Shift Selection

Version 1.0 (Beta)

  • Different image formats (8 bit indexed and 32 bit)
  • Palette Modes (Indexed, 32 bit)
  • Animate Sprites
  • Undo
  • Redo

Possible Features

  • Animation Export (.gif)
  • Voxel Image Editor
  • Project File (Program specific settings)
  • Multi Layer edit mode
