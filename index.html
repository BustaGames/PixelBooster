Pixel::Booster

A simple open source pixel art image editor

The Project

The first release of Pixel::Booster, version 0.0 alpha is out now. It is going to be an easy to use and powerful software for pixel artists. The software is being developed using C++11 and Qt5.5.1. Will feature tools used by known and approved programs used by multiple projects over the past decades, but will also contain modern features to help creating assets for games and animations with more comfort and agility. Pixel::Booster is going to be entirely free and open source forever. This means the community can improve the software by suggesting new features and fixing bugs!

Currently there are only binaries for Windows (x86 and x64). For other platforms, you have to download the source code and compile the project. Please, help us to improve the software by reporting any errors at the github page!

Planned Features

Here are some of the features currently planned for the future releases of the software. The features marked by (✓) are currently implemented in the source code. For the features on each version, check the release notes.

Current Version - 0.0 (Alpha)

  • Create new images ✓
  • Load existing images ✓
  • Transfer image from canvas to edit area ✓
  • Transfer image from edit area to canvas ✓
  • Pencil Tool ✓
  • Color Pick ✓
  • Saving image (png, jpg, bmp) ✓
  • 32 color bit image format only ✓
  • Language Options (Portuguese and English only for now) ✓
  • Setting tile size ✓
  • Toggle Transparency from pick and paste mode. ✓
  • Save app state through multiple executions. ✓
  • Basic Static Color Palette ✓

Version 0.1 (Alpha) - Planned for December 9.

  • Line Tool
  • Ellipse Tool
  • Rectangle Tool
  • Bucket Tool
  • Text Tool
  • Zoom
  • Selection Tool
  • Copy
  • Paste
  • Improved Color Palette Options (Save/Load)

Version 0.2 (Alpha)

  • Show grid
  • Show Divisions
  • Change Image Size
  • Resize
  • Shift Selection

Version 1.0 (Beta)

  • Animate Sprites
  • Undo
  • Redo

Possible Features
